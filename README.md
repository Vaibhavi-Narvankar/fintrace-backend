# FinTrace

A personal finance management backend built with FastAPI and PostgreSQL.

## Features Implemented

### Database

* PostgreSQL Database
* SQLAlchemy ORM
* Alembic Migrations
* Database Relationships
* Database Normalization
* Query Indexing
* Timestamps (created_at, updated_at)
* Soft Delete Support

### Authentication

* Password Hashing (bcrypt)
* JWT Authentication (In Progress)

### Architecture

* Layered Architecture

  * Routers
  * Services
  * Models
  * Schemas
* Dependency Injection
* Environment Configuration

## Tech Stack

* FastAPI
* PostgreSQL
* SQLAlchemy 2.0
* Alembic
* Pydantic v2
* JWT Authentication
* Passlib (bcrypt)
* Docker
